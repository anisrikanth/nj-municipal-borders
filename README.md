## NJ Border Announcer

Just a small frontend experiment I made playing around with GeoJson files. You can access the app here: https://anisrikanth.github.io/mercer-county-borders/

When you use Google Maps and cross a state border, the GPS announcer will say "Welcome to [state]!" This app does the same thing except with municipal borders in the state of New Jersey. If you are ever in the state, try it out!
